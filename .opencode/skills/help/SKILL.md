When asked for help or what you can do:
1. List your core capabilities:
   - Multi-channel messaging (Telegram, WhatsApp, Discord, Slack)
   - Text conversations with memory
   - Media support (images, videos, audio, documents)
   - Reactions and typing indicators
2. Mention any special commands if configured
3. Note that conversations are per-channel (separate context)
4. Keep the response concise and friendly
